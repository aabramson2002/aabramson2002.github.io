Description

  • This project is a joke finder. You enter a word into the searchbar and it will find jokes in it with that word. There is also a button to disable searches that are NSFW (not safe for work) and a dropdown menu to sort out which category you want the joke to be. These categories range from programming jokes, to puns, and even spooky jokes.

    • Instructions/Features

  • If you click the check mark underneath a joke, it will be highlighted yellow, indicating it's one of your favorites. Clicking the "X button" returns it back to it's original color. Clicking the "Favorites" button will take you to a different website where all of your favorite jokes are stored. By pressing the x in the favorites page, the joke dissapears and you can return back to the Joke finder.
