Description

  • This project is a recreation of the shell game. Every time you press the shuffle button, the card containing the coin, either stays or place or changes places with another card. You must guess which card has the coin after shuffling.

    • Instructions/Features

  • If you click on each card, if it is the card that was holding the coin, the card transforms back into the coin, and you write down text saying that you won the game. If not, you lose and you can try again. Clicking the try again button hides the coin back to where it was before and resets the shuffle counter to zero.

    • Current Errors

  • There aren't any major errors but the cards can begin shaking or move on top of other cards if you try spamming the shuffle button without a significant part of the animation being finished.

    • Requirements met

  • I use an ES6 class within my code called Card. This class is used to store the DOM element representing the card, it's current position which is either at 0, 200, or 400 pixels from the left, and the position the card will ultimately move to.
